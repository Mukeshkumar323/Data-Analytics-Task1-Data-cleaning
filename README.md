# Task 1: Data Cleaning and Preprocessing

## Objective
The objective of this task is to clean a raw dataset by handling missing values, removing duplicate records, correcting data formats, and preparing the data for analysis.

## Dataset
- Dataset Name:Online Sales Data
- Source: Kaggle
- Total Records: 241
- Total Columns: 9

## Tools Used
- Microsoft Excel
- Python (Pandas)

## Data Cleaning Steps
1. Checked for missing values.
2. Filled or removed missing values.
3. Removed duplicate records.
4. Corrected inconsistent text values.
5. Removed extra spaces.
6. Standardized date format.
7. Renamed column headers.
8. Corrected data types.

## Results
- Missing values handled successfully.
- Duplicate rows removed.
- Date format standardized.
- Dataset cleaned and ready for analysis.

## Files Included
- sales_data.csv (Raw Dataset)
- cleaned_sales_data.csv (Cleaned Dataset)
- data_cleaning.ipynb (Python Notebook)
- README.md

## Conclusion
This task improved the quality of the dataset by fixing common data issues. The cleaned dataset is now ready for data analysis and visualization.
